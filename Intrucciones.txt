En este video explico un ejemplo básico de proceso ETC carga incremental de datos que es aplicado a la realidad en el campo de inteligencia de negocios. 
El ejercicio consiste en cargar una base de datos destino(una base de datos multidimencional) con información recién ingresada en una base de datos origen (base de datos relacional). 
Ambas bases de datos son prácticamente la misma a diferencia que la de destino está en función de dimensiones.
La cantidad de productos ingresados en la base de datos origen deben cargarse previa ejecución del proyecto a una base de datos destino
